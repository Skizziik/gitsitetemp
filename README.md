# Serramenti Brescia

Sito vetrina per **Serramenti Brescia** — serramenti, infissi, porte, tapparelle e zanzariere su misura a Brescia.

Redesign statico in stile Apple: tipografia ampia, molto spazio bianco, navigazione translucida con blur, animazioni allo scroll e palette neutra con accento ambra del brand.

## Struttura

- `index.html` — pagina singola (hero, prodotti, perché noi, lavori, dove siamo, contatti)
- `styles.css` — design system e layout responsive
- `script.js` — nav allo scroll, menu mobile, reveal animations
- `assets/img/` — logo e foto prodotti
